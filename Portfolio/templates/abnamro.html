Project voor ABN AMRO

In het tweede half jaar van het eerste jaar kreeg ik voor het eerst een opdracht van een opdrachtgever. De opdracht van deze opdrachtgever was het maken van een gebruikersgedrag analyse systeem. Hiervoor moesten we eerst zelf een eigen site maken, omdat de regels van de bank erg streng zijn. Op deze site hebben we het gedrag van de gebruikers kunnen analyseren. Op deze manier kon de opdrachtgever inspiratie opdoen uit onze code en bleef alles veilig.

Rol binnen het project

Dit project heb ik samen met één iemand anders gedaan. In dit project heb ik alle niet-technische producten eerst zelf gemaakt. Daarna liet ik ze goedkeuren door mijn teamgenoot. Op deze manier kwamen we uiteindelijk op mooie producten. Alle technische producten deden we samen. We verdeelde aan het begin van de sprint de taken voor het coderen. Dit was eigenlijk altijd helft helft. Hierdoor konden we beide deze leerkomsten goed bewijzen. Daarnaast was dit project heel erg uitdagend op het gebied van coderen, dus zo leerde we allebei heel erg veel.
